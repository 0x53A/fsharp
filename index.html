F# Compiler (open source edition) @ github

The F# compiler, core library and core tools (open source edition). It uses the Apache 2.0 license.

What is it

The open source edition of the F# compiler helps you use F# across platforms wih Mono and MonoDevelop, particularly on Mac/OSX, Linux, Android and iOS. On Windows you normally use the Visual F# tools from Microsoft.

Contribute

Please help make using F# better on OSX, Linux, Android ad iOS. Don't ask for permission to contribute, just fork the project on Github, commit your changes and send a pull request to https://github.com/fsharp/fsharp.
