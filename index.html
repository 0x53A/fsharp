Contribute

Please help make using F# better on OSX, Linux, Android and iOS.

Clone with:

$ git clone git://github.com/fsharp/fsharp

Don't ask for permission to contribute, just fork the project on Github, commit your changes and send a pull request to https://github.com/fsharp/fsharp.
